#나의 할 일 목록 (My Todo App)

간단하고 현대적인 할 일 관리 앱입니다.

## 기능

- 할 일 추가/삭제
- 완료/미완료 토글
- LocalStorage 데이터 저장
- 중복 방지
- 필터링 (전체/진행중/완료)
- 완료 항목 일괄 삭제

## 사용 방법

1. 할 일을 입력하고 추가 버튼 또는 Enter 키를 누르세요
2. 체크박스로 완료 여부를 토글할 수 있습니다
3. 삭제 버튼으로 개별 항목을 삭제할 수 있습니다
4. 필터 버튼으로 항목을 필터링할 수 있습니다
5. 완료된 항목을 한 번에 삭제할 수 있습니다
